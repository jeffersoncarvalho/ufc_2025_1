TRABALHO FINAL COM CARÁTER EXTENSIONISTA
DESENVOLVIMENTO DE SOFTWARE PARA A WEB

- EQUIPE DE NO MÁXIMO 5 PESSOAS
	- DEFINIR O CARGO DE CADA UMA E SUAS RESPONSABILIDADES;
	- DEFINIR UMA SOFTWARE DE CONTROLE DE VERSÃO (GITHUB E/OU GITLAB)

- SOBRE O TEMA
	- O TEMA DEVE SER EXTENSIONISTA
		- ESCOLAS PÚBLICAS DO MUNICÍPIO OU ESTADO
		- ASSOCIAÇÕES DE FEIRANTES, COMERCIANTES
		- SOLUÇÕES PARA O MUNICÍPIO (CLIENTE PREFEITURA)
			- SECRETARIA DA SAÚDE
			- SECRETARIA DO ESPORTE
			- ETC.
		- SOLUÇÕES DE INFRAESTRUTURA
		- SOLUÇÕES DE COMBATE ZOONOSES 
		- SOLUÇÕES PARA O ABANDONO/ADOÇÃO DE ANIMAIS
		- GUIAS TURÍSTICOS
		
	- A EQUIPE DEVE TER UM CLIENTE DA SOCIEDADE PARA CONVERSAR
	- REGISTRO FOTOGRÁFICO (SUBIR AS FOTOS NO CONTROLE DE VERSÃO)

- SOBRE AS EXIGÊNCIAS TÉCNICAS

	- O SITE DEVE TER, INICIALMENTE, UM PROTÓTIPO (EX. FIGMA)
	- IMPLEMENTAR O SITE USANDO HTML SEMÂNTICO, CSS E JAVASCRIPT.
	- OPCIONAL (IMPLEMENTAR USANDO ALGUMA BIBLIOTECA OU FRAMEWORKS)
		- REACT
		- ANGULAR
		- VUE
		- BOOTSTRAP
		- MUI
		- TAILWIND
		- ETC.
	- O SITE DEVE TER NO MÍNIMO 5 PÁGINAS DE CONTEÚDO (NÃO INCLUE LOGIN, CADASTRO)
	- O SITE DEVE SE COMUNICAR COM ALGUMA API
		- ALGUMA JÁ PRODUZIDA OU IMPLEMENTADA PELA EQUIPE
		- OS DADOS DO SITE, EM SUA MAIORIA, DEVEM SER ALIMENTADOS VIA O CONSUMO DA API.
		- O CONSUMO DA API DEVE SER FEITO USANDO FETCH OU AXIOS
	- A INTERFACE DEVE SER RESPONSIVA
